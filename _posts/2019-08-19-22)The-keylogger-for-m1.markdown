## 22.1) 악성 키로거의 목표  
* 가장 큰 목표는 **돈**이다.  

# 22.1.1) 온라인 게임  
* 매년 거래되는 게임 아이템 거래 금액은 일반인들의 상상을 초월할 정도인데, 각 사용자들의 계정 정보를 키로거를 통해 획득하면 실제 현금 거래량은 상상을 초월한다.  

# 22.1.2) 인터넷 뱅킹  
* 은행 계정을 악의적인 목적은 가진 자가 획득한다면 이것 또한 온라인 게임과 같은 효과를 불러 일으킨다.  

# 22.1.3) 기업 정보 유출  
* 이쪽은 하드웨어 키로거와도 연관성이 있는데, 내부에 공조자가 있다면 정보 유출의 가능성이 훨씬 높아지며, 피해 규모는 상상을 초월한다.  
* 직원들의 올바른 윤리관이 요구되는 사안이다.  

## 22.2) 키로거의 종류와 향후 발전 방향  
* 하드웨어 키로거: 작은 USB 메모리처럼 생겨서 키보드 케이블 끝에 연결하여 PC에 연결 가능  
* 향후에 키보드를 대체할 I/O 장치가 나타난다면(음성 인식, 필기 도구) 그에 맞춰진 키로거 또한 나올 것  

## 22.3) 그렇다면 대처는 어떻게?  
* 키로거 대응 행동 수칙  
1. 공공장소의 PC에서는 개인정보 입력 금지  
2. 백신 업데이트는 항상 최신으로!  
3. 개인 정보 입력에 Copy(Cut) & Paste를 활용해볼 것  
4. 개인 방화벽 사용할 것  

## 22.4) 개인정보  
* 정보화 시대에서 개인정보는 **그 사람을 대신할 수 있는 정보**이다.  
* 비밀번호는 자신만이 알 수 있도록(권장: 숫자 + 문자 + 특수문자, 8자리 이상) 설정할 것.
